Nette vouchers
=================
Simple voucher activation system based on nette **(half-completed project)**

### Clone

```bash
  git clone https://github.com/revlenuwe/nette-vouchers.git
  cd nette-vouchers
```

### Server Setup

The simplest way:

	php -S localhost:8000 -t www
